# HK Wealth Projector - 香港投资回报计算器

![HK Wealth Projector](https://img.shields.io/badge/Streamlit-1.32.0+-brightgreen)
![Python](https://img.shields.io/badge/Python-3.8+-blue)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

一个基于 Streamlit 构建的香港投资回报模拟工具，支持多语言切换，可模拟不同资产类别（现金存款、保险储蓄计划、香港房地产、全球股票）在不同时间周期内的投资回报表现。

## ✨ 核心功能

### 📊 资产类别模拟
- **现金/银行存款**：基于年化利率的固定收益模拟
- **保险储蓄计划**：支持保守型/进取型两种产品模型，考虑回本周期
- **香港房地产**：综合资本增值率和净租金收益率
- **全球股票（标普500）**：基于历史波动率的蒙特卡洛模拟

### 🌐 多语言支持
- 简体中文 / 繁體中文 / English 三语言界面
- 所有文本、标签、提示均支持语言切换

### 📈 可视化分析
- 资产增长轨迹对比图表（Plotly 交互式图表）
- 关键指标卡片展示最终收益和增长率
- 可查看底层数据表格并导出 CSV

### 🎛️ 灵活配置
- 自定义初始投资额（港币）
- 调整投资年限（0-100年）
- 精细调节各类资产的收益率、波动率等参数

### 📱 界面特性
- 硅谷风格紧凑化 UI 设计
- 响应式布局，适配不同屏幕尺寸
- 侧边栏参数配置，主界面结果展示
- 访问计数器统计

## 🚀 快速开始

### 环境要求
- Python 3.8 或更高版本
- 所需依赖包：

```bash
pip install streamlit pandas numpy plotly scipy
```

### 运行应用

1. 将代码保存为 `hk_wealth_projector.py`
2. 执行以下命令启动应用：

```bash
streamlit run hk_wealth_projector.py
```

3. 浏览器访问 `http://localhost:8501` 即可使用

## ⚙️ 配置说明

### 侧边栏参数
| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 初始投资额 | 模拟的初始投资金额（港币） | 1,000,000 |
| 投资年限 | 模拟的投资周期 | 50年 |
| 银行存款年利率 | 现金类资产的年化收益率 | 3.5% |
| 保险产品 A IRR | 保守型保险的内部收益率 | 4.2% |
| 保险产品 A 回本年限 | 保守型保险的回本周期 | 6年 |
| 保险产品 B IRR | 进取型保险的内部收益率 | 6.5% |
| 保险产品 B 回本年限 | 进取型保险的回本周期 | 9年 |
| 房地产资本增值率 | 香港房产的年度增值率 | 3.0% |
| 房地产净租金收益率 | 扣除成本后的租金回报率 | 2.5% |
| 股票平均年回报率 | 标普500的年化收益率 | 8.5% |
| 股票年度波动率 | 股票收益的波动幅度 | 14.0% |

### 计算逻辑
- **现金存款**：简单复利计算 `FV = PV * (1 + r)^n`
- **保险产品**：回本周期内线性回本，回本后复利增长
- **房地产**：综合资本增值和租金收益的复利计算
- **股票**：基于几何布朗运动的随机模拟

## 📊 输出结果

### 指标卡片
- 各类资产在投资期末的最终价值（百万港元）
- 相对于初始投资的增长率百分比

### 可视化图表
- 交互式折线图展示各类资产随时间的增长轨迹
- 悬停显示具体年份的资产价值
- 可切换/隐藏不同资产类别曲线

### 数据表格
- 完整的年度资产价值数据
- 支持 CSV 格式导出

## 🎨 界面定制

### 样式调整
- 修改 CSS 中的颜色、字体、间距等参数可自定义界面风格
- 调整图表高度、颜色映射可改变可视化效果
- 可修改列布局比例优化显示效果

### 多语言扩展
1. 在 `LANG_DICT` 中添加新的语言键值对
2. 更新语言选择器的选项
3. 确保所有文本都有对应的翻译

## ⚠️ 免责声明

本工具仅用于教育模拟目的：
- 保险收益曲线为简化模型，仅考虑回本周期，不代表具体保单演示
- 房地产收益假设租金再投资且不包含交易成本
- 股票波动基于历史数据模拟，不构成投资建议
